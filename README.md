## Social Media Website - Tarmeez

This project is a social media website called Tarmeez.
 It allows users to register, log in, create posts, edit posts, delete posts, view profiles
 and interact with other users by commenting on their posts.

### Technologies Used
- HTML
- CSS (Bootstrap, Bootstrap Icons, Font Awesome)
- JavaScript (axios)

### Features
- User authentication: Users can register and log in securely.
- Create/Edit/Delete posts: Users can create, edit, and delete their own posts.
- View profiles: Users can view profiles and see posts made by other users.
- Commenting: Users can comment on posts made by other users.

### File Structure
- **index.html**: Main HTML file containing the structure of the website.
- **style/style.css**: CSS file for styling the website.
- **scripts/main.js**: Main JavaScript file containing global functions and setup.
- **scripts/index.js**: JavaScript file specifically for the index page functionality.
- **fontawesome-free-6.5.1-web/**: Directory containing Font Awesome icons.
- **node_modules/**: Directory containing Bootstrap and axios libraries.
- **images/**: Directory containing images used in the project.

### Setup Instructions
1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to access the website.

### How to Use
1. Register or log in using the provided forms.
2. Once logged in, you can create a new post by clicking the "Create Post" button.
3. You can also edit or delete your existing posts by clicking on the respective buttons on each post.
4. View profiles by clicking on the username of any post.
5. Log out using the "Log Out" button.
